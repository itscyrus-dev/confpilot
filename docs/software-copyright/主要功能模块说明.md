# ConfigPilot 主要功能模块说明

## 一、首页仪表盘模块

显示 GitHub 授权状态、同步仓库、配置文件数量、最近同步时间和自动监听状态，为用户提供整体运行概览。

## 二、授权模块

通过 GitHub Device Flow 获取授权码、展示登录地址、轮询授权结果，并在授权成功后保存 token。

## 三、配置扫描模块

识别 zsh 与 Ghostty 配置路径，展示文件存在状态和目标仓库路径，为同步操作提供基础数据。

## 四、同步模块

提供立即备份、从云端恢复、双向同步三个入口。备份过程包含文件复制、manifest 生成、Git commit 和 Git push。恢复过程包含远端拉取、本地备份和配置还原。

## 五、自动监听模块

监听配置文件变化，对连续变更进行防抖处理，并自动触发备份同步。

## 六、冲突处理模块

展示冲突配置文件、本地哈希、远端哈希，并提供使用本地、使用远端和打开文件处理三类操作。

## 七、设置模块

展示应用数据目录、本地同步仓库路径、GitHub 用户、仓库状态，并提供清理本地缓存功能。
