# ConfigPilot 源代码目录说明

## 一、根目录

- `package.json`：前端依赖、Tauri 命令脚本和项目元数据。
- `vite.config.ts`：Vite 开发服务器与构建配置。
- `tsconfig.json`：TypeScript 编译配置。
- `README.md`：项目介绍、运行方式和安全说明。

## 二、前端目录

- `src/main.tsx`：React 应用入口，包含页面导航、状态加载、Tauri command 调用和主要界面组件。
- `src/styles.css`：应用样式文件，定义布局、面板、表格、日志、按钮和状态样式。

## 三、Tauri 后端目录

- `src-tauri/Cargo.toml`：Rust 依赖、包信息和构建配置。
- `src-tauri/tauri.conf.json`：Tauri 应用窗口、构建和打包配置。
- `src-tauri/build.rs`：Tauri 构建脚本。
- `src-tauri/src/main.rs`：Tauri 桌面应用入口。
- `src-tauri/src/lib.rs`：核心后端逻辑，包含配置扫描、GitHub 授权、仓库创建、同步、恢复、监听、冲突处理、状态存储和 Tauri commands。

## 四、软著文档目录

- `docs/software-copyright/软件说明书.md`
- `docs/software-copyright/用户操作手册.md`
- `docs/software-copyright/系统设计说明书.md`
- `docs/software-copyright/主要功能模块说明.md`
- `docs/software-copyright/源代码目录说明.md`

该目录用于辅助软件著作权申请材料整理。
