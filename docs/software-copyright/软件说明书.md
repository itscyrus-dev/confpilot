# ConfigPilot 个人配置文件同步管理系统软件说明书

## 一、软件名称

ConfigPilot 个人配置文件同步管理系统。

## 二、软件概述

ConfigPilot 是一款面向个人开发者和终端用户的桌面端配置文件同步软件。软件基于 Tauri 2 框架开发，后端采用 Rust 语言实现核心同步、安全存储、GitHub 授权和文件监听能力，前端采用 React 与 TypeScript 构建图形化界面。

软件主要解决个人电脑中 zsh 与 Ghostty 配置文件分散、迁移困难、多设备同步不便和手动备份容易遗漏的问题。用户通过 GitHub 授权后，软件自动创建私有配置仓库，并将本地配置文件同步至云端仓库，实现配置备份、恢复、监听同步和冲突保护。

## 三、主要功能

1. 配置文件扫描：自动识别 zsh 与 Ghostty 常用配置路径，展示文件存在状态、类型、哈希值和更新时间。
2. GitHub 授权：通过 GitHub Device Flow 完成桌面端授权，不要求用户在软件内输入 GitHub 密码。
3. 私有仓库管理：自动检查并创建默认私有仓库 `configpilot-dotfiles`。
4. 配置备份：将本地配置复制到同步工作区，生成 manifest 文件，并提交推送到 GitHub。
5. 云端恢复：从远端仓库恢复配置文件到本机，恢复前保留本地备份。
6. 自动监听：监听配置文件变化，合并短时间内的频繁保存操作，自动触发备份。
7. 冲突处理：本地与远端同时变化时生成本地和远端冲突副本，由用户选择保留策略。
8. 状态展示：展示授权状态、仓库状态、同步状态、配置列表、冲突列表和运行日志。

## 四、运行环境

- 操作系统：macOS
- 桌面框架：Tauri 2
- 后端语言：Rust
- 前端框架：React、TypeScript、Vite
- 云端服务：GitHub

## 五、软件特点

ConfigPilot 采用本地优先的同步设计，所有配置文件仍保留在用户原始路径。软件不直接上传高风险敏感目录，GitHub token 使用系统安全存储保存。同步冲突不会静默覆盖用户文件，确保个人配置安全可控。
